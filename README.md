# Business Intelligence Data Pipeline

End-to-end BI pipeline project covering relational data modeling, SQL querying, Power Pivot integration, and Power BI reporting as part of a comprehensive final exam.

## Skills Demonstrated
- SQL query development and data extraction
- Entity-relationship diagram (ERD) design
- Power BI report building and data visualization
- Power Pivot multi-table data modeling
- Data export and transformation workflows

## Tools & Technologies
- SQL, Power BI, Power Pivot, Excel, Tableau (.tflx)

## Contents
- `question_4_ww.sql` — SQL queries for final exam question 4
- `final exam question 4 output.csv` / `part b output.csv` — query result exports
- `final exam question 5 ERD.png` — entity-relationship diagram
- `question 5 part b dataframe output.csv` — structured output data
- `bsan_6060_final_q5_fact_v2.tflx` — Tableau data source file
- `question 6 power bi.pdf` — Power BI dashboard export
- `how_to_get_multiple_tables_in_powerpivot.docx` — Power Pivot reference guide
